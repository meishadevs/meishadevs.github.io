使用对象展开运算符合并对象

两个对象存在同名的字段时，后面对象的字段值会覆盖前面对象的字段值，这种操作只会将前一个对象的属性赋值给后面对象的属性

let obj1 = {
  name: 'meishadevs',
  age: 25,
  sex: '男'
}

let obj2 = {
  name: '小红',
  age: 21,
  sex: '女',
  hobby: '唱歌'
}

let obj3 = { ...obj1, ...obj2 }
